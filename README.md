NomadNest

NomadNest is a full-stack web application built with Node.js, Express, MongoDB, and EJS. It allows users to create, view, edit, and delete property listings. The project demonstrates RESTful routing, CRUD operations, and dynamic rendering using EJS templates.

Technologies Used

Backend: Node.js, Express.js

Database: MongoDB, Mongoose

Frontend: EJS, HTML, CSS

Middleware: Method-Override (for PUT and DELETE requests)

Installation and Setup

Clone the repository

git clone https://github.com/yourusername/NomadNest.git
cd NomadNest


Install dependencies

npm install


Start MongoDB
Ensure that MongoDB is running locally at:

mongodb://127.0.0.1:27017/nomadnest


Run the application

node app.js


The server will start at:
http://localhost:8080
