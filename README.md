<h1 align="center">🏡 NomadNest</h1>

<p align="center">
  <b>NomadNest</b> is a full-stack web application built with <b>Node.js</b>, <b>Express</b>, <b>MongoDB</b>, and <b>EJS</b>.  
  It allows users to create, view, edit, and delete property listings.  
  The project demonstrates <b>RESTful routing</b>, <b>CRUD operations</b>, and <b>dynamic rendering</b> using <b>EJS templates</b>.
</p>

---

## ⚙️ Technologies Used

- **Backend:** Node.js, Express.js  
- **Database:** MongoDB, Mongoose  
- **Frontend:** EJS, HTML, CSS  
- **Middleware:** Method-Override (for PUT and DELETE requests)

---

<h2>🚀 Installation and Setup</h2>

<ol>
  <li><strong>Clone the repository</strong>
    <pre><code>git clone https://github.com/yourusername/NomadNest.git
cd NomadNest</code></pre>
  </li>

  <li><strong>Install dependencies</strong>
    <pre><code>npm install</code></pre>
  </li>

  <li><strong>Start MongoDB</strong>
    <p>Ensure that MongoDB is running locally at:</p>
    <pre><code>mongodb://127.0.0.1:27017/nomadnest</code></pre>
  </li>

  <li><strong>Run the application</strong>
    <pre><code>node app.js</code></pre>
  </li>
</ol>

<p>The server will start at:<br>
<strong>http://localhost:8080</strong></p>
